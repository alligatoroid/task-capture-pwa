# Todo Items

This folder contains individual todo items captured from the Task Capture PWA inbox.

## Structure

Each todo item is a separate markdown file with the following format:
- Filename: `YYYY-MM-DD-title.md`
- Contains: Task description, creation date, source, and status

## Processing Workflow

1. Items are captured in Task Capture PWA inbox
2. Inbox items are extracted and saved here as individual todo files
3. Items are organized and prioritized
4. Completed items are moved to the Done section
